## Análise do Manual do Usuário do Sistema UDVNMG

### Observações Iniciais:
- O sistema é chamado 'Real Valor - Sistema Financeiro UDV'.
- A tela de login possui campos para CPF e Senha.
- O manual é um PDF com capturas de tela do sistema.

### Funcionalidades Relevantes (baseado nas imagens e títulos):
- **Recebimento:** Parece ser a área para registrar pagamentos.
  - Há campos para 'Nome', 'Tipo' (Mensalidade, F.P.S.G., F.R., F.S., Plantio), 'Ano', 'Mês' (jan a dez), 'Vlr. Unit.' e 'Confirma'.
  - Há uma tabela de 'RECIBO' com colunas para 'entrada', 'código', 'ano' e meses.
  - Botões: 'Consultar Mensalidades', 'Fechar Recibo', 'Impar', 'Utilizar recibo reservado'.
  - A emissão de recibo parece gerar um pop-up com o recibo.
- **Inclusão:** Possivelmente para incluir novos associados ou dados.
- **Pagamento:** Não detalhado ainda, mas pode ser para registrar pagamentos a fornecedores, etc.
- **Consulta:** Provavelmente para consultar histórico de pagamentos ou débitos.
- **Impressão:** Para imprimir relatórios ou recibos.
- **Outros:** Funções diversas.
- **Sair:** Para sair do sistema.
